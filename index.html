My Projects

Project 1: TapNTab

TapNTab is an Android app that streamlines nightlife experiences by allowing users to open, track, and close tabs seamlessly using NFC technology for quick check-ins and payments. Built with Kotlin for the front end and Firestore for real-time backend support, it ensures to update deals and menus for individual clients. Customers avoid long waits, while bars benefit from faster service and increased sales. The app’s cashless, contactless approach enhances convenience, making it the ultimate smart tab management solution for bars, restaurants, and nightlife venues.

Key Learning:

Through building TapNTab, I’ve gained key insights into seamless NFC-based transactions, real-time Firestore data synchronization, and scalable Kotlin architecture for Android. Optimizing bar workflow automation while ensuring low-latency payments has deepened my understanding of user-centric design, database efficiency, and fintech integration, making nightlife transactions frictionless and data-driven.

Project 2: mixdup Music Collaboration

Currently working on Mixdup, a music collaboration platform, is an open-source platform designed for musicians to collaborate on music projects using a workflow inspired by GitHub. Users can create repositories (songs), open issues (tasks), fork projects, submit collab requests (pull requests), and track contributions with version control. The frontend is being built with React, providing a dynamic and responsive user experience, while Supabase will be used to power the backend, database management, and file storage for seamless collaboration. This system enables musicians to work on tracks efficiently, track changes, and attribute contributions, making remote music production structured, transparent, and community-driven.

Key Learning:

TThrough this project, I seek to have gained deeper insights into version-controlled collaboration for music production, leveraging React for an interactive UI and Supabase for scalable backend operations. Adapting GitHub’s workflow to audio files, handling forking, branching, and contribution tracking, and ensuring seamless multi-user editing has sharpened my understanding of structured creative workflows.
