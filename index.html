My Projects

Project 1: Dynamic Lift Experiment

This is a research project I have been working on at the College of Engineering at UW-Madison under Professor Robert Radwin and PhD student Zihao Zhu.
This project aims to analyze daily labor movements and optimize posture and movements in order to maximize output in high-movement jobs. I have worked collaboratively on different datasets and have prepared and processed over 25TB of data which is to be trained through a model provided by Lightbuzz. I am currently working on training the AI motion-based model.

Key Learning:

The primary objective of this project was to acquire insights into motion-based and posture tracking, as well as to deepen my understanding of Lightbuzz. The essence of this project closely mirrored that of PostureX, making it crucial to gain valuable experience from more experienced individuals in the field.

Project 3: Teeko AI

I developed a Python-based implementation of the Teeko game, integrating an AI player using a minimax algorithm with heuristic evaluation for optimal move selection. This project simulates the classic board game, allowing for an engaging player vs. AI experience. The AI evaluates potential moves up to a specified depth to ensure competitive gameplay. The project was built using Python, leveraging libraries such as NumPy for efficient state comparison and game logic implementation.
